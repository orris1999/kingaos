日期：2026-05-13

本报告只记录 Finance dry-run 页面的结构识别验收结果。验收对象是财务提交的 8 份报价表文件，处理方式为本地只读结构识别；不上传文件、不写数据库、不保存 dry-run 结果、不导入报价表、不生成报价草稿、不生成正式报价。

## 安全边界

- 报价表 / 成本表 / 价格候选数据由财务提交和维护。
- 出口部不能上传或维护价格表。
- dry-run 只识别文件结构、sheet、表头候选、adapter 匹配、字段映射和风险提示。
- 检测到价格 / 成本列时，只显示“成本候选列：是 / 否”“报价候选列：是 / 否”。
- 本报告不记录具体 KJ 行、OEM 行、成本金额、报价金额、价格决策信息或完整 Excel 内容。
- 价格候选不是财务批准价格，正式报价必须后续接 FinancePricing。

## 风险提示验收

页面风险提示必须继续显示：

- 成本价不是财务批准价格。
- dry-run 不生成正式报价。
- dry-run 不自动批准价格。
- 报价表由财务提交和维护。
- 出口部不能上传或维护报价表。

水箱 / 中冷器匹配到 adapter 时，页面必须显示：

- 水箱 / 中冷器存在多编码、多规格、多包装字段，V1 需要行级人工确认。

特殊包装及其他匹配到 adapter 时，页面必须显示：

- 特殊包装及其他只能作为包装 / 附加项候选，不能作为产品标准报价。
